Job Description

We are looking for an experienced backend operations engineer to maintain our production and testing environments.

Responsibilities
• Configure and deploy our server clusters in the AWS cloud environment.
• Maintain multiple clusters for development, testing, and production.
• Automate the configuration and deployment of these clusters.
• Manage and move data among the multiple environments.
• Understand the cost and performance tradeoffs.

Requirements
• Substantial programming experience with a language such as C++, Python, Node.js, or Go.
• Configuring and deploying Amazon Web Services and programming with its API. Understand the cost model. EC2, S3, and others.
• CentOS Linux administration and tools. Command line experience, shell scripting, networking and firewalls, ssh, and kernel configuration.
• SSL/TLS experience with OpenSSL.
• Server cluster configuration and deployment.
• Server performance monitoring and tuning.
• MongoDB configuration and deployment.
• MongoDB queries and performance tuning.
• 3+ years of backend operations experience
• US Citizen

Nice to Haves
• C/C++ programming experience.
• SQL experience.
• Other cloud services, such as Google Cloud.
• Virtual machines.
• Docker container configuration and deployment.

Skills
Python, C++, SQL, MongoDB, Node.js, C, Centos, Server Cluster Management, Virtual Machines, AWS/EC2/ELB/S3/DynamoDB, OpenSSL, Docker, Google Cloud Platform, Server and Network Monitoring and Management, Go (programming language), SSL/TLS
